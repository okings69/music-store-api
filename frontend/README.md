# Music Store Frontend

React/Vite frontend for the Music Store Showcase project.

## Features

- seed-based song browsing
- table view with expandable details
- gallery view with infinite scrolling
- language selection
- likes slider
- audio preview playback

## Local development

```powershell
cd frontend
copy .env.example .env
cmd /c npm install
cmd /c npm run dev
```

Open:

- `http://localhost:5173`

## Environment variable

```env
VITE_API_URL=http://localhost:5000/api
```
